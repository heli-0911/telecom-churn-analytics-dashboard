# AI Powered Telecom Customer Churn Analytics Dashboard

This project analyzes telecom customer behavior and predicts churn using machine learning.

## Features

* Interactive dashboard built with Streamlit
* Customer churn prediction using Random Forest
* KPI metrics and churn insights
* Interactive visualizations with Plotly
* Feature importance analysis
* Correlation heatmap
* Real-time churn probability prediction

## Tech Stack

Python
Streamlit
Scikit-Learn
Pandas
Plotly
Seaborn

## Dataset

IBM Telecom Customer Churn Dataset (7043 customers)

## How to Run

Install dependencies

pip install -r requirements.txt

Run the dashboard

streamlit run app.py

## Output

Interactive AI dashboard that helps telecom companies identify customers likely to churn.
